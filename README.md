# 🚀 QuickGPT 
### AI Chat • Image Generation • Credits & Payments

QuickGPT Pro is a modern **full-stack AI platform** built with the **MERN stack**, designed to deliver intelligent conversations, AI-generated images, and a secure credit-based usage system with **Stripe payments**.

The platform integrates powerful AI models like **Gemini/OpenAI** and handles media efficiently using **ImageKit**, making it scalable, secure, and production-ready.

DEMO LINK-  https://quick-gpt-zeta-ten.vercel.app/ 

## ✨ Key Highlights

### 🤖 Smart AI Chat
- Context-aware multi-message conversations  
- Persistent chat history  
- High-speed AI responses  
- Supports Gemini & OpenAI models  

### 🎨 AI Image Creation
- Generate high-quality AI images  
- Automatic uploads via ImageKit  
- Image previews inside chat  
- Image history storage  

### 🔐 Authentication & Security
- JWT-based user authentication  
- Secure password hashing with bcrypt  
- Protected routes using middleware  

### 💰 Credit-Based Usage System
- Credits required for text & image generation  
- Real-time credit deduction  
- Credit balance managed at user level  

### 💳 Stripe Payment Integration
- Buy credits using Stripe Checkout  
- Secure transaction handling  
- Webhook-based credit updates  
- Reliable & verified payment flow  

### 💬 Chat Organization
- Create and delete chat threads  
- Multiple conversations per user  
- Clean and structured chat management  

### 🛠️ Scalable Backend Design
- Modular controller-service architecture  
- Mongoose data models  
- Environment-based configuration  
- Production-ready REST APIs
